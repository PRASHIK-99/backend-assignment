# Backend + Frontend Assignment

This project contains:
- Backend (Node.js, Express, MongoDB)
- Frontend (React + Vite + Tailwind)
- JWT Authentication
- Role-based Access
- CRUD APIs for Tasks
- Swagger API Docs at `/api-docs`
- Scalability notes included

## How to Run Backend
cd backend
npm install
cp .env.example .env
npm run dev

## How to Run Frontend
cd frontend/frontend
npm install
npm run dev
